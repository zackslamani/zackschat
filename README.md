### Temo Chat for Ionic 2
Theme for ionic 2
### Pages
* Login
* Chat room
* List friends
* My Profile
* Profile friend

### How to run ?
You need npm and ionic installed
* 1- Download or clone this repo

* 2- In folder of project run
 ```shell
 npm install
  ```
* 3 Run ionic 
 ```shell
 ionic serve
  ```
  The new tab in your favorite navigator will open
  
 * 4 Enjoy
